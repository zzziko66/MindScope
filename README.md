MindScope - 思维透视
智能化解析网页内容，识别文本中的说服手段、情绪操控、认知偏差与隐性信息
MindScope 是一款轻量化浏览器扩展程序。本产品依托人工智能技术，对网页文本进行实时解析，拆解内容中的营销逻辑、情绪诱导方式及认知陷阱，量化信息可靠程度，辅助用户理性甄别网络信息，规避信息误导。
✨ 核心功能
- 🎯 说服技巧分析：系统性解析文本所使用的营销手段与说服逻辑，梳理内容创作底层逻辑。
- 🔥 情绪操纵识别：识别文本内含的情绪渲染、刻意煽动、焦虑诱导等操控手段，规避情绪化误导。
- 🔍 隐性信息挖掘：甄别文本中刻意省略、模糊表述、隐性引导的关键信息，还原完整信息维度。
- 🧠 认知偏差检测：精准识别幸存者偏差、锚定效应、从众心理等常见认知谬误，辅助用户建立客观判断思维。
- 📊 综合可信度评估：基于多维度分析指标生成量化可信度评分，直观判定网页内容可靠性等级。
💻 安装教程（Chrome 通用）
本扩展兼容全部基于Chromium内核的浏览器（Chrome、Edge、夸克等），采用本地离线部署方式，部署流程简洁规范：
1. 克隆或下载本仓库源码，解压至本地目录（建议路径不含中文、空格及特殊字符）。
2. 浏览器地址栏输入：chrome://extensions/，进入扩展程序管理页面。
3. 开启页面右上角开发者模式。
4. 点击左上角加载已解压的扩展程序按钮。
5. 选中解压完成后的 mindscope 根目录文件夹，完成部署安装。
⚙️ 项目配置
本程序依托大语言模型接口实现智能化分析功能，用户需自行申请合法有效的AI接口密钥，当前兼容以下主流大模型服务：
- OpenAI（GPT 系列模型）
- DeepSeek（深度求索）
- Anthropic Claude 系列模型
用户可在扩展配置界面填写接口密钥并选择对应模型，保存配置后即可启用全部分析功能。
